# Glossary (Canonical Terms)

cost_factor: construction reality multiplier from City, State
market_factor: pricing power / demand multiplier from City, State

NOI assets: value primarily via cap rate
Operating assets: value primarily via cash flow/payback

DSCR: debt service coverage ratio
LTC: loan-to-cost
Yield on cost: NOI / total project cost
RevPAR: Revenue per Available Room
ADR: Average Daily Rate
GOP: Gross Operating Profit
